# Site Images

Place construction, project, team, logo, and favicon assets under this folder.
Anything under `public/` is served from the site root, so a file at
`public/images/projects/site-photo.webp` is referenced as
`/images/projects/site-photo.webp`.

## Folders

| Folder | Use |
| ------ | --- |
| `favicon/` | Browser icons and manifest files |
| `hero/` | Large hero or banner imagery |
| `logo/` | Brand logo and wordmark assets |
| `sections/` | Project, service, and content-section imagery |

Use lowercase, hyphenated filenames and keep image alt text updated in
`src/content/siteText.ts`.
