# Template Usage

## Run

```bash
npm install
npm run dev
npm run build
```

## Edit Content

- Copy and structured lists live in `src/content/siteText.ts`.
- Navigation lives in `src/content/navigation.yml`.
- Public images live under `public/images/`.

## Deploy

The source branch is `main`. The generated GitHub Pages branch is `gh-pages`.

```bash
bash deploy.sh
```
