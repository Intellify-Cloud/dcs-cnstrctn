# Design Notes

The visual system follows `docs/DESIGN.md`: industrial, corporate, precise, and
site-first.

## Core Rules

- Use deep navy for major structural surfaces.
- Use safety yellow sparingly for actions, highlights, and status emphasis.
- Use concrete neutrals for background layers.
- Keep cards, buttons, inputs, and sections sharp-edged.
- Use Montserrat for headlines, Inter for body text, and IBM Plex Sans for
  technical labels.
